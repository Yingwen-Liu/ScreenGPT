# ScreenGPT
Auto ChatGPT on the area of the screen you selected


- setArea.py: set the area to be read
- readArea.py: read the text on the selected area. Method read_area returns the OCR text
- run.py
- config.json: configurations
	- "area": the area selected to be sent to GPT. Press UP to reset the area
	- "path": directory to Tesseract OCR
